# 🔐 Secure File Encryption and Storage System

## 📌 Overview
This project is a Secure File Encryption and Storage System developed using Python.  
It helps users protect sensitive files by encrypting them and storing them securely with user authentication.

---

## 🎯 Objectives
- Protect user data from unauthorized access
- Provide secure login system
- Encrypt and decrypt files safely
- Store user data using database

---

## ⚙️ Features
✅ User Registration & Login  
✅ Password Authentication  
✅ File Encryption  
✅ File Decryption  
✅ Secure Storage using Database  
✅ Error Handling (Invalid login, etc.)

---

## 🛠️ Technologies Used
- Python
- SQLite (Database)
- File Handling
- Basic Encryption Logic
